Longform template

A boilerplate for longform journalism articles. Made for designers learning responsive webdesign and developers wanting a streamlined workflow.


By James Benedict
This template is meant for people of all skill levels and aims to allow designers to spend more time designing and less time coding. Tweaks and changes are encouraged as it is only the boilerplate for you to build on. You can view some examples of stories developed with this version of the template here and older versions here.

Alignment

link to stand alone example

There are nine layout classes to get started with and the css is found in alignment.css.

Inline

Inline is the main column on the screen and where this text is now. It takes up 60% of the screen, with a 20% margin on either side. To go wider use the wide or full classes. You can also divide the inline column up internally with aside.

Asides

There are three type of asides, two that go inside the inline column and one that does not. For each type, there are two sepearte classes for right and left. The two inline are aside class="left" and aside class="left-wide". The aside class="left" placed outside of the inline column will float to the edge of the screen.

Wide

The wide class is usually used for media or graphics. It has a width of 75% and a margin-left of 15%. You can either place a figure inside a wide div or apply the class to the figure itself.

full

The full class is a fullscreen alignment best used for media.

Leads

There are three types of leads, lead-text, lead-img and lead-overlay. You can also create your own using ai2html. All styles are in the media.css. Since these usually go at the top of a page view the standalone html examples to see how they look normally.
