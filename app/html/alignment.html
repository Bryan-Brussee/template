link to stand alone example

There are six places you can use for layout and the css is found in alignment.css. Click here for a version of this page with background colors for all elements. This will help get a sense of how they respond to each other.

Inline

Inline is the main column on the screen. You can use it as a wrapper or go wider with the wide or full classes. You can also divide it up internally with aside.

Wide class

The wide class is usually used for media or graphics. You can either place a figure inside a wide div or apply the class to the figure itself.

full

The full class is a fullscreen alignment best used for media.
